A small standalone launcher for Path of Exile 1 and 2. 
It lets you start the game, launch selected tools and mods, open useful websites, and manage separate settings for each game from one simple interface.

It will start any mods you have added to it and the close mods button will stop any mods that was startet with the launcher. (it will not stop mods that was not started by the launcher)
if you add labcompass you can use the button to start it whenever you need it

To add mods you can just drop the mod files into the mod settings or find path to it.
websites youi add will open in your default browser
